## 1ページ目：これは何？

### FormBoost AI営業社員 とは

**「上長は判断するだけ」の営業DXツール** を開発中です。

- 550万社の企業データベースから、毎日AIが「今日送るべきリスト」を自動提案
- 送付する文章もAI（Claude Haiku 4.5）が会社ごとに自動生成
- Slackに「これで送っていい？」と通知 → **承認ボタン1つで全自動送信**
- 問い合わせフォームURLが無くても、HPクロール → AI検索の3段階で自動発見
- 実行前に「今回は◯◯円かかります」とコスト事前見積もり

### 何がうれしいか

| Before（今の営業） | After（FormBoost AI） |
|---|---|
| リスト作成に半日 | AIが毎朝提案 |
| 1通ずつコピペ送信 | ワンクリックで一括送信 |
| 文面作成に悩む | 会社ごとに自動生成 |
| 結果の振り返りが属人的 | AIが昨日の結果を分析→改善提案 |

## 2ページ目：1日の流れ

### 朝（自動）
1. AIが昨日の送付結果を分析
2. 今日のターゲット候補を選定（業種・地域・資本金などで絞り込み）
3. 会社ごとに差別化ポイントを入れた文面を自動生成
4. **Slackに「今日の営業プラン」として通知**

### 昼（あなたは判断するだけ）
- Slackで「✅ 承認」「✏️ 編集」「❌ 却下」のいずれかを押す
- 承認 → 裏側で FormBoost が自動で全件送信
- 送信後、Cloud Run + Playwright がフォームを自動入力

### 夕方（自動）
- 送付結果をDBに記録
- **添付したURLをクリックした会社を「ホットリード」として自動リストアップ**
- 明日のAIブリーフィングに反映

## 3ページ目：技術スタック & 進捗

### 技術構成
- **フロント**: Next.js 16 / React 19 / Tailwind
- **バックエンド**: Next.js API Routes / Supabase (PostgreSQL)
- **AI**: Claude Haiku 4.5（文面生成・日次ブリーフィング）
- **送信基盤**: Cloud Run + Playwright
- **通知**: Slack Block Kit + Interactivity
- **探索**: Brave Search API（フォームURL自動発見）
